## 남은 기록 공백

다음 항목은 상세 데이터가 없으므로, 향후 밸런스나 UI 개선 판단이 필요하면 별도 보강한다.

- 회차별 진행 세대 수
- 회차별 플레이 시간
- 회차별 종료 대표작위와 종료 유형
- 가장 의미 있었던 선택
- 가장 이해하기 어려웠던 선택
- UI에서 자주 확인한 정보
- UI에서 찾기 어려웠던 정보
- 작은 창/세로형 화면의 구체 문제 여부
