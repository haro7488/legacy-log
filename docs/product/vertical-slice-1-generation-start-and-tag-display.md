# Vertical Slice 1 세대 시작 및 유산 태그 표시 기준

> 최종 수정: 2026-05-19

## 목적

이 문서는 Vertical Slice 1에서 세대 시작 화면이 이전 세대의 결과를 어떻게 보여주고, 활성 유산 태그를 어느 정도까지 노출할지 정리한다.

목표는 플레이어가 새 세대 시작 시 "왜 이 상태에서 시작하는가"를 빠르게 이해하고, 동시에 과도한 정보 표시로 피로해지지 않게 하는 것이다.

## 적용 범위

- 세대 시작 화면의 정보 우선순위
- 활성 유산 태그 최대 노출 개수
- 태그 노출 우선순위
- 태그별 세대 시작 문장 템플릿
- 작은 창과 세로형 화면에서의 표시 기준

## 제외 범위

- 최종 UI 레이아웃
- 폰트, 색상, 아이콘 스타일
- 데이터 파일 형식
- 태그 정렬 알고리즘 세부 구현
- 최종 문체 확정

## 세대 시작 화면의 역할

세대 시작 화면은 보상 화면이 아니라 연결 화면이다.

플레이어는 이 화면에서 다음을 이해해야 한다.

1. 이전 세대가 무엇을 남겼는가
2. 이번 세대가 어떤 조건에서 시작하는가
3. 이번 인물이 어떤 성향과 강점/약점을 가지는가
4. 이번 세대에서 무엇을 조심하거나 노릴 수 있는가

이 정보는 10초 안에 훑어볼 수 있어야 한다.

## 기본 정보 구성

세대 시작 화면은 다음 순서를 권장한다.

| 순서 | 정보 | 목적 |
|---:|---|---|
| 1 | 가문명, 대표작위, 세대, 새 인물 | 현재 플레이 주체 확인 |
| 2 | 이전 세대 한 줄 요약 | 직전 결과 기억 |
| 3 | 핵심 유산 태그 | 다음 세대에 영향을 주는 흔적 확인 |
| 4 | 이번 세대 출발 상태 | 재산, 명예, 궁정 영향력, 결속의 질적 구간 확인 |
| 5 | 후계자/가주 특성 | 이번 세대 운영 차이 확인 |
| 6 | 이번 세대의 압력 또는 기회 | 사건 풀 방향 예고 |

## 활성 태그 노출 개수

기본 화면에서 한 번에 보여주는 활성 유산 태그는 최대 4개로 제한한다.

권장 구성은 다음과 같다.

- 가문사 태그 최대 1개
- 세대 태그 최대 2개
- 단기 태그 최대 1개

활성 태그가 4개를 넘으면 보조 영역에서 전체 목록을 확인하게 한다. 기본 화면은 이번 세대 사건 풀에 가장 직접적으로 영향을 주는 태그만 보여준다.

## 태그 노출 우선순위

활성 태그가 많을 때는 다음 우선순위로 고른다.

1. 멸문, 작위 상실, 후계 불안과 직접 연결되는 태그
2. 이번 세대 사건 풀을 바꾸는 태그
3. 직전 세대의 큰 사건에서 생긴 태그
4. 대표작위와 직접 연결되는 태그
5. 상태 변화만 설명하는 단기 태그

예를 들어 `몰락의 징조`, `불안한 후계`, `박탈 위기`는 `상인 후원`보다 먼저 보여준다. 반대로 재산형 태그가 이번 세대 핵심 압력일 때는 `가문의 부채`나 `풍요로운 영지`가 우선될 수 있다.

## 태그 표시 문구 기준

태그는 내부 키처럼 보이면 안 된다. 화면에는 짧은 라벨과 한 줄 설명을 함께 쓴다.

| 내부 태그 | 화면 라벨 예시 | 한 줄 설명 방향 |
|---|---|---|
| `왕의 총애` | 왕의 총애 | 왕실 보상과 청원 기회가 열려 있다. |
| `궁정의 의심` | 궁정의 의심 | 궁정 사건에서 방어가 필요하다. |
| `정적의 표적` | 정적의 표적 | 정치적 공격과 탄핵 위험이 높다. |
| `왕실의 채무` | 왕실의 빚 | 왕실 의존이 다음 선택의 부담이 된다. |
| `전쟁 공훈` | 전쟁 공훈 | 명예와 승격 명분으로 쓰일 수 있다. |
| `기사의 명예` | 기사의 명예 | 명예 사건에서 신뢰를 얻기 쉽다. |
| `불명예` | 불명예 | 평판 회복 사건이 필요하다. |
| `겁쟁이의 소문` | 겁쟁이의 소문 | 전쟁과 명예 사건에서 약점으로 작용한다. |
| `풍요로운 영지` | 풍요로운 영지 | 재산 기반이 안정적이다. |
| `피폐한 영지` | 피폐한 영지 | 영지 불안과 탄원 사건이 늘어난다. |
| `상인 후원` | 상인 후원 | 재산 기회와 외부 의존이 함께 온다. |
| `가문의 부채` | 가문의 부채 | 작위 유지와 재산 사건의 압박이 커진다. |
| `안정된 후계` | 안정된 후계 | 세대 전환이 안정적이다. |
| `불안한 후계` | 불안한 후계 | 후계권 공격과 멸문 위험이 커진다. |
| `내부 균열` | 내부 균열 | 친족과 가신단 갈등이 커진다. |
| `충성스러운 친족` | 충성스러운 친족 | 내부 협력과 섭정 기회가 열린다. |
| `승격 명분` | 승격 명분 | 상위 작위 청원 기회가 가까워진다. |
| `불안한 승격` | 불안한 승격 | 새 작위가 아직 안정되지 않았다. |
| `박탈 위기` | 박탈 위기 | 대표작위 상실 위험이 크다. |
| `변경의 책무` | 변경의 책무 | 전쟁과 방위 부담이 늘어난다. |
| `피의 원한` | 피의 원한 | 복수와 내부 폭력 사건의 불씨가 남았다. |
| `잊히지 않는 패배` | 잊히지 않는 패배 | 전쟁 실패가 평판과 선택을 압박한다. |
| `가문의 맹세` | 가문의 맹세 | 결속과 장기 목표의 근거가 된다. |
| `몰락의 징조` | 몰락의 징조 | 멸문과 쇠락 위험이 가까워진다. |

라벨 문구는 최종 문체 확정 전까지 임시 기준이다. 단, UI 검증 빌드에서는 라벨이 무엇을 의미하는지 플레이어가 이해할 수 있어야 한다.

## 세대 시작 문장 템플릿

세대 시작 문장은 태그를 나열하지 않고, 이번 세대의 출발 조건으로 풀어쓴다.

템플릿은 다음 구조를 따른다.

`이전 세대의 [결과/사건]은 [태그 의미]로 남았다. 이번 세대는 [기회/위험] 속에서 시작한다.`

태그별 예시는 다음과 같다.

| 태그 | 세대 시작 문장 예시 |
|---|---|
| `왕의 총애` | 이전 세대의 충성은 왕의 기억에 남았다. 이번 세대는 궁정의 문이 조금 더 열린 상태에서 시작한다. |
| `궁정의 의심` | 궁정은 아직 이 가문을 완전히 믿지 않는다. 이번 세대의 청원과 방어는 더 조심스러워야 한다. |
| `정적의 표적` | 누군가는 이 가문의 상승을 기억하고 있다. 이번 세대는 궁정의 시선 아래에서 시작한다. |
| `왕실의 채무` | 왕실과 맺은 빚은 아직 끝나지 않았다. 이번 세대는 은혜와 부담을 함께 물려받았다. |
| `전쟁 공훈` | 전장에서 얻은 이름은 아직 사라지지 않았다. 이번 세대는 그 공훈을 명분으로 삼을 수 있다. |
| `기사의 명예` | 선대의 명예로운 선택은 기사들의 입에 남았다. 이번 세대는 그 신뢰를 등에 업고 시작한다. |
| `불명예` | 선대의 오명은 쉽게 씻기지 않았다. 이번 세대는 먼저 평판을 회복해야 한다. |
| `겁쟁이의 소문` | 물러섰다는 소문은 가문보다 오래 남았다. 이번 세대는 전장과 궁정에서 그 말을 견뎌야 한다. |
| `풍요로운 영지` | 선대가 다진 영지는 아직 힘을 낸다. 이번 세대는 비교적 안정된 재산 기반에서 출발한다. |
| `피폐한 영지` | 영지는 아직 선대의 부담을 회복하지 못했다. 이번 세대는 백성의 불만과 부족한 재산을 함께 안고 시작한다. |
| `상인 후원` | 상인들은 이 가문의 가능성에 돈을 걸었다. 이번 세대는 그 후원과 이해관계 속에서 시작한다. |
| `가문의 부채` | 빚은 장부에만 남지 않았다. 이번 세대의 선택은 먼저 채권자와 유지 비용을 의식해야 한다. |
| `안정된 후계` | 후계는 공개적으로 인정받았다. 이번 세대는 권리 다툼보다 운영에 집중할 수 있다. |
| `불안한 후계` | 후계의 이름은 정해졌지만 모두가 받아들인 것은 아니다. 이번 세대는 정통성을 증명해야 한다. |
| `내부 균열` | 가문 안의 금은 아직 닫히지 않았다. 이번 세대는 친족과 가신의 불신을 안고 시작한다. |
| `충성스러운 친족` | 선대가 남긴 신뢰는 친족의 충성으로 돌아왔다. 이번 세대는 내부의 손을 빌릴 수 있다. |
| `승격 명분` | 선대의 공적은 더 높은 작위를 요구할 근거가 되었다. 이번 세대에는 청원의 기회가 보인다. |
| `불안한 승격` | 작위는 올랐지만 기반은 아직 따라오지 못했다. 이번 세대는 새 지위를 지켜내야 한다. |
| `박탈 위기` | 대표작위는 아직 남아 있지만, 그 아래의 땅은 흔들린다. 이번 세대는 상실을 막아야 한다. |
| `변경의 책무` | 변경을 맡은 이름은 명예이자 짐이다. 이번 세대는 전쟁과 방위의 압력을 피하기 어렵다. |
| `피의 원한` | 피로 끝난 선택은 살아남은 자들의 기억에 남았다. 이번 세대는 복수와 불신의 씨앗을 안고 시작한다. |
| `잊히지 않는 패배` | 패배는 전장에서 끝나지 않았다. 이번 세대는 그 기억을 딛고 다시 이름을 세워야 한다. |
| `가문의 맹세` | 선대가 남긴 맹세는 아직 가문을 묶고 있다. 이번 세대는 그 약속을 지키거나 배신해야 한다. |
| `몰락의 징조` | 지난 세대의 끝은 쇠락의 예고처럼 남았다. 이번 세대는 멸문을 피할 길을 찾아야 한다. |

## 여러 태그가 있을 때 문장 구성

여러 태그가 있을 때는 모든 태그를 문장으로 풀지 않는다.

- 기본 화면은 가장 중요한 1~2개 태그만 문장화한다.
- 나머지 태그는 라벨과 한 줄 설명으로 보조 영역에 둔다.
- 서로 상반되는 태그가 있으면 최신 태그만 문장화한다.
- 긍정 태그와 부정 태그가 함께 있으면 부정 태그를 먼저 보여준다. 플레이어가 이번 세대의 위험을 먼저 이해해야 하기 때문이다.

## 작은 창/세로형 화면 기준

작은 창과 세로형 화면에서는 다음 순서를 유지한다.

1. 가문/작위/세대/인물 헤더
2. 핵심 태그 2~4개
3. 이전 세대 한 줄 요약
4. 이번 세대 압력
5. 자세히 보기 탭

화면이 좁을수록 문장을 줄이고, 태그 라벨과 변화 방향을 우선한다.

## 수동 검증 질문

- 새 세대가 왜 이 상태에서 시작하는지 10초 안에 이해되는가?
- 핵심 태그 4개 제한이 충분한가, 부족한가?
- 가장 위험한 태그가 기본 화면에 먼저 보이는가?
- 태그 문장이 사건 결과와 자연스럽게 연결되는가?
- 작은 창이나 세로형 화면에서 정보가 너무 길게 밀리지 않는가?
- 보조 영역을 열지 않아도 다음 선택을 시작할 수 있는가?
