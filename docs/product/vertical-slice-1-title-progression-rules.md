## 목적

이 문서는 Vertical Slice 1에서 작위 상승과 작위 상실이 어떤 제품 기준으로 발생해야 하는지 정리한다.

목표는 작위를 단순 레벨이 아니라 **기회와 위험을 동시에 여는 귀족가문의 사회적 지위**로 작동시키는 것이다. 정확한 수치, 확률, 데이터 구조, 구현 알고리즘은 이 문서에서 정하지 않는다.

## 적용 범위

Vertical Slice 1에서 실제 플레이 영향이 있는 작위는 다음 범위다.

1. 남작
2. 자작
3. 백작
4. 후작
5. 공작

왕과 황제는 세계관 압력 또는 장기 목표로 암시할 수 있지만, 실제 통치 시스템과 등극 시스템은 이번 범위에서 제외한다.

## 작위의 세 가지 기능

작위는 다음 세 가지 기능을 동시에 가진다.

| 기능 | 설명 | 플레이 영향 |
|---|---|---|
| 사회적 지위 | 가문이 어떤 위상으로 불리는지 정한다. | UI, 연대기, 세대 요약의 대표 호칭 |
| 기회 | 더 높은 사건, 보상, 궁정 접근을 연다. | 승격, 궁정 정치, 전쟁 공훈, 영지 기회 |
| 위험 | 유지 비용, 정적, 박탈, 반역 의심을 키운다. | 작위 상실, 쇠락, 멸문 위험 |

작위 상승은 항상 좋은 결과가 아니다. 높은 작위는 더 큰 선택지를 열지만, 더 큰 정치적·재정적 압박을 만든다.

## 대표작위 기준

가문은 여러 작위를 동시에 가질 수 있다.

Vertical Slice 1에서는 구현 복잡도를 줄이기 위해, 플레이어에게 보여주는 기본 호칭은 **대표작위**로 정리한다.

대표작위 기준은 다음과 같다.

- 보유 작위 중 가장 높은 작위를 기본 대표작위로 삼는다.
- 대표작위가 상실되면 남은 작위 중 가장 높은 작위가 새 대표작위가 된다.
- 작위 일부 포기나 박탈이 발생하면, 결과 화면과 세대 요약에서 대표작위 변화를 명시한다.
- 여러 작위 보유의 세부 목록은 보조 정보 또는 가문사 상세로 보낼 수 있다.

## 작위별 기본 압력

| 대표작위 | 기본 플레이 성격 | 주요 기회 | 주요 위험 |
|---|---|---|---|
| 남작 | 생존과 기반 확보 | 영지 안정, 명예 회복, 첫 승격 명분 | 재산 부족, 영지 불안, 낮은 궁정 접근 |
| 자작 | 지역 영향력 확대 | 시장권, 소규모 궁정 기회, 백작령 기회 | 비용 증가, 인접 귀족 견제 |
| 백작 | 궁정과 전쟁의 중심 진입 | 왕의 총애, 전쟁 공훈, 백작령 확장 | 정적, 징발, 정치적 의심 |
| 후작 | 변경과 군사 책무 | 변경 방위, 후작위 명예, 공작 승격 발판 | 변경의 책무, 전쟁 위험, 박탈 위기 |
| 공작 | 왕권 근처의 거대 권력 | 대형 정치 사건, 왕실 보상, 장기 목표 암시 | 반역 의심, 파벌 견제, 작위 상실 시 큰 쇠락 |

## 승격 기회 조건

승격은 자동으로 발생하지 않는다. 조건을 만족하면 **승격 기회 사건**이 등장하고, 플레이어가 그 기회를 받아들일지 선택한다.

승격 기회는 다음 조건을 조합해 발생한다.

- 현재 대표작위
- 핵심 상태
- 유산 태그
- 최근 큰 사건 결과
- 세대 진행 정도

## 작위별 승격 기준

| 승격 | 필요한 제품 기준 | 대표 사건 | 주요 비용/위험 |
|---|---|---|---|
| 남작 -> 자작 | 영지 기반 또는 명예 기반이 충분해야 한다. | `PRO-01 자작위 청원` | 재산 소모, `불안한 승격`, 정적 관심 |
| 자작 -> 백작 | 궁정 영향력, 왕의 총애, 전쟁 공훈 중 하나가 강해야 한다. | `PRO-02 백작의 빈 영지` | `정적의 표적`, 재산 부담, 결속 흔들림 |
| 백작 -> 후작 | 변경 방위나 큰 전쟁 공훈이 있어야 한다. | `PRO-03 변경 방위 공적` | `변경의 책무`, 전쟁 사건 증가, 재산 압박 |
| 후작 -> 공작 | 장기 권력 기반, 궁정 영향력, 계승 명분이 필요하다. | `PRO-04 공작의 후계 단절` | `정적의 표적`, `불안한 승격`, 반역 의심 위험 |

## 승격 비용 원칙

승격 선택은 다음 비용 중 하나 이상을 동반해야 한다.

- 재산 소모: 청원, 예물, 의전, 새 영지 정비
- 가문 결속 하락: 친족 간 권력 다툼, 후계 경쟁
- 궁정 위험 증가: 정적, 의심, 파벌 연루
- 명예 손상 가능성: 무리한 청원, 거래, 매수
- 유산 태그 부여: `불안한 승격`, `정적의 표적`, `왕실의 채무`, `변경의 책무`

좋은 승격 사건은 플레이어에게 "지금 올라가도 버틸 수 있는가?"를 묻게 해야 한다.

## 작위 상실 조건

작위 상실은 즉시 게임 오버가 아니라 멸문 전 쇠락 단계다.

작위 상실 위험은 다음 조건에서 생긴다.

- 재산이 장기간 낮고 `가문의 부채`가 남아 있을 때
- 명예가 낮고 `불명예` 또는 `겁쟁이의 소문`이 남아 있을 때
- 궁정 영향력이 낮고 `궁정의 의심`, `정적의 표적`, `박탈 위기`가 있을 때
- 가문 결속이 낮고 `불안한 후계`, `내부 균열`이 있을 때
- `몰락의 징조`가 남은 상태에서 큰 위기 사건을 맞을 때

## 작위 상실 방식

작위 상실은 다음 방식 중 하나로 표현한다.

| 상실 방식 | 의미 | 플레이 영향 |
|---|---|---|
| 대표작위 상실 | 가장 높은 작위를 잃는다. | 대표작위 하락, 세대 요약에 강하게 반영 |
| 하위 작위만 유지 | 가문은 남지만 위상이 낮아진다. | 사건 풀 축소, 승격 재도전 가능 |
| 영지 상실 | 특정 영지 기반이 무너진다. | 재산/결속 악화, `피폐한 영지` 가능 |
| 작위 유지 비용 증가 | 아직 잃지는 않았지만 부담이 커진다. | 재산 사건 압박, `가문의 부채` 가능 |
| 후계 권리 약화 | 다음 세대 승계가 흔들린다. | `불안한 후계`, 멸문 위험 |
| 폐위/추방 | 현 세대 인물이 퇴장한다. | 세대 종결, `몰락의 징조` |

## 작위 위험 단계

UI에서는 작위 위험을 정확한 숫자보다 단계 라벨로 보여준다.

| 단계 | 의미 | UI 설명 방향 |
|---|---|---|
| 안정 | 작위 유지에 큰 위협이 없다. | "작위 안정" |
| 견제 | 정적이나 비용 부담이 보인다. | "작위 견제" |
| 위기 | 박탈/하락 사건이 가까워진다. | "작위 위기" |
| 박탈 위협 | 방어하지 못하면 대표작위 상실 가능성이 크다. | "박탈 위협" |

단계 문구는 최종 UI 문체 라운드에서 조정할 수 있다.

## 작위 관련 유산 태그 연결

| 태그 | 작위 영향 |
|---|---|
| 승격 명분 | 다음 승격 기회 사건의 조건 또는 보정으로 사용한다. |
| 불안한 승격 | 승격 직후 위기 사건, 정적 사건, 유지 비용 사건의 조건으로 사용한다. |
| 박탈 위기 | 작위 상실 사건의 직접 조건으로 사용한다. |
| 변경의 책무 | 후작 이상에서 전쟁/변경 사건 우선도를 높인다. |
| 왕의 총애 | 승격 기회와 왕실 보상 사건 조건으로 사용한다. |
| 궁정의 의심 | 반역 혐의, 탄핵, 후계권 무효 청원 조건으로 사용한다. |
| 정적의 표적 | 궁정 공격, 탄핵, 계승 분쟁 조건으로 사용한다. |
| 왕실의 채무 | 왕실 요구, 징발, 후속 청원 비용으로 되돌아온다. |
| 몰락의 징조 | 작위 상실이 멸문 위험으로 이어지는 조건으로 사용한다. |

## UI/UX 표시 기준

작위 관련 정보는 다음 위치에서 보여준다.

### 항상 보이는 정보

- 대표작위
- 작위 위험 단계가 위기 이상일 때 경고 1개

### 사건 화면

- 이 사건이 승격 기회인지, 작위 위기인지 명확히 표시한다.
- 관련 작위 조건 또는 유산 태그를 짧게 보여준다.

예:

- 사건 성격: 승격 기회
- 관련 유산: `승격 명분`
- 위험: 정적 증가 가능

### 결과 화면

- 대표작위 변화
- 작위 위험 단계 변화
- 새 작위 관련 유산 태그
- 작위 상실 또는 유지의 이유

### 세대 시작 화면

- 현재 대표작위
- 직전 세대의 작위 변화
- 현재 작위 위험 단계
- 작위와 연결된 핵심 유산 태그 1~2개

## 세대 요약 반영 기준

- 승격이 발생한 세대는 세대 요약에 반드시 반영한다.
- 대표작위 상실 또는 박탈 위기가 발생한 세대도 반드시 반영한다.
- 작위 위험이 높았으나 방어에 성공한 경우도 가문사의 중요한 사건으로 기록할 수 있다.
- 작위 변화가 없더라도 `불안한 승격`, `변경의 책무`, `왕실의 채무`처럼 다음 세대에 영향을 주는 작위 태그는 요약 후보가 된다.

## 남은 Root 기획 작업

- 작위 위험 단계별 세대 시작 문장 템플릿 작성
- 승격/상실 사건의 선택지별 연대기 변형 문장 작성 여부 결정
- 작위 조건과 세대별 사건 풀 선택 기준 연결
- 작위 상승/상실 후 상태 변화량의 제품 기준 구체화
- 작위 UI 수동 검증 항목 작성
